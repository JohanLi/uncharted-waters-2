About

This is a remake of Uncharted Waters: New Horizons, an open world RPG and simulation game released in 1994 and developed by Koei. The game is set during the Age of Exploration, dealing with sailing and trading.

The remake is far from complete and merely a side project of mine, so any progress made will be sporadic. Visit my blog for updates. The code for this game is available on GitHub.
